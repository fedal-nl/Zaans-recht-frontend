# Zaans-recht-frontend
A simple html/css website for zaans-recht

## CLI Application

This repository also includes a command line interface application built with Python 3.13, InquirerPy, and Rich. The CLI provides an interactive interface for legal information and services.

### Quick Start with Docker

```bash
cd cli
docker build -t zaans-recht-cli .
docker run -it zaans-recht-cli
```

### Features

- Interactive menu system using InquirerPy
- Beautiful terminal output with Rich formatting
- Legal information display
- Document search functionality
- Contact information access

For detailed CLI documentation, see [cli/README.md](cli/README.md).
